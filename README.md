[# QGis-Plugin LayerLib #

QGis-Python-Plugin to
- store metadata for map-container-files (*.qgz, *.qgs, *.qlr)
- show registered map-container-files in TreeView
- add registered sub-layers to current project


## Addendum ##
- The plugin has been developed under the latest versions since 2026, currently
  - QGIS 3.44.7 'Solothurn'
  - QGIS 4.02 'Norrköping'
  - Windows (10 + 11)
  - Linux (Ubuntu/Mint 21.2)
- not tested (but should run) with older QGis-3-Versions
- not tested on macOS
- please report bugs or ideas for missing features
- or translation-errors :-)
